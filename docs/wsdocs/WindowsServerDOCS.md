# AD域控

## 服务器配置

在`服务器管理器`里点击`添加角色和功能`然后在`服务器角色`选项卡选择`Active Directory域服务`和`DNS服务器`(在添加DNS服务器之前先把网络从DHCP改成Static)

<img src="./添加域控功能.png" alt="添加域控功能" style="zoom:200%;" />

在安装完成后点击`将此服务器提升为域控制器`

![点击提升为域控制器](./点击提升为域控制器.png)

在打开的部署配置页面点击`添加新林`然后输入根域名

![输入根域名](./输入根域名.png)

在域控制器选项页面设置`还原模式(DSRM)密码`

![输入DSRM密码](./输入DSRM密码.png)

在其他选项页面系统会自动生成NetBIOS域名,生成后点击下一步即可

在先决条件检查这一步，如果没有设置密码会被验证失败，可按下`Ctrl + Alt + Delete`后点击`修改密码`(这只能修改当前用户)

如果你使用的是Server用户,则按下`WIn + R`在运行窗口里输入`control`在打开的控制面板窗口点击`用户账户`,`管理账户`,`更改账户`,选择`Administrator`用户再修改密码

![修改密码](./修改密码.png)

先决条件检查通过后就点击安装,安装完成后会重启服务器

重启后可右键点击`此电脑`的`属性`，然后查看系统属性(不同的版本可能会有差异)里的`域`是否为配置后的`域`名称，如果是则配置成功。

![域名查看](./域名查看.png)

配置完后先添加一个用户，在`服务器管理器`中点击`工具`里的`Active Directory 用户和计算机`

![点击工具里的Active Directory用户](./点击工具里的Active Directory用户.png)

在打开的新窗口中右键点击域文件夹中的`新建`里的`用户`

![点击新建用户](./点击新建用户.png)

在打开的新建用户窗口中填写好用户信息后点击下一步

![填写新建用户信息](./填写新建用户信息.png)

在这个页面可以填写好，可勾选`用户下次登陆时须更改密码`或密码永不过期(如果不勾选密码永不过期则默认的过期时间是42天)，完成后点击下一步然后点完成

![密码配置](./密码配置.png)

那么这个时候可以配置客户机加入到`AD域`当中了

## 客户机配置

打开你的客户机系统,按下`Win + R`在打开的运行窗口输入`control`然后点击`网络和Internet`里的网络和共享中心，点击你的网卡

![客户机点击网卡](./客户机点击网卡.png)

在打开的窗口中点击`属性`，双击`Internet协议版本4(TCP/IPv4)`,然后点击`使用下面的DNS服务器地址`,将刚刚在服务器上配置的Static IP地址输进去后按确定

![ip配置](./ip配置.png)

然后右键`此电脑`的`属性`点击`系统属性`(不同的版本可能会有差异)里的`计算机名`，点击`更改`按钮，打开新窗口后，点击`域`，然后输入域名后点确定

![更改域名](./更改域名.png)

> [!NOTE]
>
> 提示:如果连接不上并带有下列域控制器由查询识别,但是无法连接到域控制器之类的字眼，尝试把服务器上的Windows安全中心里的防火墙全部关掉

弹出登录窗口后输入在服务器上添加的用户的用户名和密码

![输入账号密码](./输入账号密码.png)

点击确认后成功加入到域中则需要重启电脑

重启后可右键单击`此电脑`里的`属性`中的`系统属性`查看`域`名称判断是否加入成功(这个过程需要登陆Administrator用户)

# 组策略管理

打开在`服务器管理器`中的`工具`里的`组策略管理`

![打开组策略管理](./打开组策略管理.png)

然后点击当前`域`名称下的`Default Domain Policy`里的设置，即可打开默认的组策略

![默认策略管理](./默认策略管理.png)

然后右键点击当前`域`中的编辑，即可修改组策略

![打开组策略编辑](./打开组策略编辑.png)

## 用户安全策略

![用户安全策略](./用户安全策略.png)

在此次可编辑用户的密码，账户锁定等常用策略

## 设置网络共享(SMB协议)并映射为网络驱动盘

在服务器上右键点击需要共享的文件夹，点击属性，在打开的窗口里点击共享，在打开的`高级共享`窗口中勾选`共享此文件夹`并按需设置共享名称，随后设置权限(可按需设置权限),完成后点击确定

![共享文件夹操作](./共享文件夹操作.png)

这时你的共享路径为你的`\\计算机名称\共享名`，我这里的是`\\WIN-M4EPOLT051B\Shared`

随后打开组策略编辑窗口，在`用户配置`->`首选项`->`Windows设置`->`驱动器映射`中，右键点击空白窗口在弹出的菜单中点击`新建`->`映射驱动器`

在新打开的窗口中将`操作`设置为创建，在`位置`里填写共享路径，随后按需设置盘符，完成后点击确定

![映射网络驱动器设置](./映射网络驱动器设置.png)

在客户机中验证策略是否设置成功

![网络共享映射客户机验证](./网络共享映射客户机验证.png)

## 下发文件夹和文件

打开`组策略编辑中`的`用户配置`->`首选项`->`Windows设置`里的文件夹，随后右键点击空白部分，在弹出的菜单中点击`新建`->`文件夹`

![下发文件夹之新建](./下发文件夹之新建.png)

在打开的新窗口中将`操作`选择为`创建`，`路径`填写的是客户机的路径，切勿填写为本机服务器的路径。完成后点击确定

![下发文件夹之新建文件夹窗口](./下发文件夹之新建文件夹窗口.png)

随后打开Windows设置里的`文件`，打开后右键点击空白部分，在弹出的菜单中点击`新建`->`文件`

![下发文件之新建](./下发文件之新建.png)

在打开的新窗口中将`操作`选择为`创建`，`源文件`填写一个客户机可以访问到的路径(建议在这个操作之前先部署SMB服务)，`目标文件`填写的是客户机的路径，切勿填写为本机服务器的路径(注意:这里一定要在文件夹目录后面填写目标文件的文件名，不要只填写目标文件夹，否则策略不会生效)。完成后点击确定

![下发文件之新建文件窗口](./下发文件之新建文件窗口.png)

在客户机中验证策略是否设置成功

![验证文件下发策略结果](./验证文件下发策略结果.png)

![限制USB存储设备设置](./限制USB存储设备设置.png)

在客户机中验证

![验证限制USB存储设备](./验证限制USB存储设备.png)
